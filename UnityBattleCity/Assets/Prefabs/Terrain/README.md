# 地形预制体说明

## 1. 砖墙 (Wall)
- **标签**: Wall
- **碰撞**: 可被玩家和敌人的子弹摧毁
- **颜色**: 棕色
- **大小**: 1x1单位

## 2. 钢板 (Steel)
- **标签**: Steel
- **碰撞**: 不可被子弹摧毁
- **颜色**: 灰色
- **大小**: 1x1单位

## 3. 海水 (Water)
- **标签**: Water
- **碰撞**: 不可通过，坦克会被阻挡
- **颜色**: 蓝色
- **大小**: 1x1单位

## 4. 森林 (Forest)
- **标签**: Forest
- **碰撞**: 可通过，但坦克会被隐藏
- **颜色**: 绿色
- **大小**: 1x1单位

## 5. 基地 (Base)
- **标签**: Base
- **碰撞**: 不可被玩家子弹摧毁，但可被敌人子弹摧毁
- **颜色**: 黄色
- **大小**: 1x1单位
- **功能**: 基地被摧毁则游戏结束

## 6. 地板 (Floor)
- **标签**: Floor
- **碰撞**: 可自由通过
- **颜色**: 灰色
- **大小**: 1x1单位
